## ✨ Key Features

*   **📊 Real-Time Tracking**: Record and analyze fuel consumption with precision across all your devices.
*   **🌐 Multi-Platform Excellence**: One app, everywhere. Enjoy premium aesthetics and a native-feel experience on iPhone, Android, browser, and desktop.
*   **🌍 Multi-lingual Support (9 Languages)**: Fully localized in: **English, Greek, German, French, Spanish, Italian, Russian, Chinese, and Hindi**.
*   **🔐 Biometric Security**: Protect your data with FaceID/TouchID or Fingerprint, accompanied by an automatic Privacy Screen for ultimate privacy.

---

## 🛠️ Technical Excellence

For those who value quality and architecture, **Fuel Operation Tracker** is built with the most modern technologies in mobile development:

*   **🏗️ Clean Architecture**: Strict separation into 4 layers (Domain, Application, Presentation, Infrastructure) for maximum flexibility and scalability.
*   **⚡ Riverpod State Management**: Utilizing `riverpod_generator` for safe and efficient state management, ensuring a bug-free experience.
*   **💾 Offline-First Experience**: Integrated **SQLite (sqflite)** for fast data access and storage without requiring an internet connection.
*   **🌙 Dynamic Theming**: Full Dark & Light mode support using `CupertinoDynamicColor` for comfortable use in all lighting conditions.
*   **🚀 Premium UX Effects**: Glassmorphism influences in UI elements and targeted Haptic Feedback for a tactile user experience.

## 🛠️ Τεχνική Αρτιότητα

Για τους λάτρεις της ποιότητας και της αρχιτεκτονικής, το **Fuel Operation Tracker** έχει χτιστεί με τις πιο σύγχρονες τεχνολογίες στον χώρο του mobile development:

*   **🏗️ Clean Architecture**: Αυστηρός διαχωρισμός σε 4 επίπεδα (Domain, Application, Presentation, Infrastructure) για μέγιστη ευελιξία και επεκτασιμότητα.
*   **⚡ Riverpod State Management**: Χρήση `riverpod_generator` για ασφαλή και αποδοτική διαχείριση της κατάστασης, διασφαλίζοντας μια bug-free εμπειρία.
*   **💾 Offline-First Experience**: Ενσωμάτωση **SQLite (sqflite)** για ταχύτατη πρόσβαση και αποθήκευση δεδομένων χωρίς την ανάγκη σύνδεσης στο internet.
*   **🌙 Dynamic Theming**: Πλήρης υποστήριξη Dark & Light mode με χρήση `CupertinoDynamicColor` για ξεκούραστη χρήση σε όλες τις συνθήκες φωτισμού.
